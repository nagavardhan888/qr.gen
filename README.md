# QR Code Generator

A simple project to generate QR codes for text, URLs, or any data you provide.  
Built to make sharing information quick and easy.

## Features
- Generate QR codes instantly
- Support for text, links, and other data
- Lightweight and easy to use
- Export QR codes as images

## Installation
Clone the repository:
```bash
git clone https://github.com/your-username/qr.gen.git
```
